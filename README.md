# VuePress 学习笔记

![](https://img.shields.io/apm/l/vim-mode)

## Clone Item

```bush
git clone git@github.com:Knowledge-Precipitation-Tribe/learning-Javascript.git
```
## Install Dependencies

```bush
npm install
```

## Serve Reload

```bush
npm run dev
```

## Build

```bush
npm run build
```

## Deploy

```bush
npm run d
```
